> Chrome Extension + Web Dashboard + Node.js Backend &nbsp;|&nbsp; Dec 2025 – Present

A **full-stack spaced repetition system** that captures DSA problems directly from LeetCode/Codeforces and auto-schedules revisions — so you stop forgetting patterns right before interviews.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)
![Manifest V3](https://img.shields.io/badge/Manifest_V3-4285F4?style=flat-square&logo=googlechrome&logoColor=white)

- ⚡ **Automated tracking** — eliminates manual logging with 1-click capture via injected Shadow DOM UI
- 📶 **Offline-first sync** — local storage with automatic backend synchronization on reconnect
- 🛡️ **Data integrity** — backend deduplication prevents corrupt revision history
- 🧠 **Spaced repetition engine** — CLEAN / SLOW / FORGOT ratings with dynamic scheduling intervals
- 📊 **Analytics dashboard** — GitHub-style heatmap, streak tracking & difficulty mastery clusters
- 🔔 **Smart notifications** — Chrome Service Worker alarms, timezone-aware, anti-spam logic
